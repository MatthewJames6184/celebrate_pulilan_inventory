**Language Stack**:
**Laravel** – backend/PHP framework
**Inertia.js** – connects Laravel to React without building a separate API (see useForm, Head from @inertiajs/react)
**React + TypeScript** – frontend (.tsx, typed interfaces like LoginForm)
**Tailwind CSS** – all the utility classes (flex, grid, gap-2, etc.)
**shadcn/ui** – your Button, Input, Label, Checkbox components (@/components/ui/...)
**lucide-react** – icon library (LoaderCircle)
**Vite** – bundler (you already knew this one)

This file matches Laravel's official React + Inertia starter kit (laravel/breeze or laravel new --react), which already scaffolds login/register/auth pages like this one.
